This folder contains a simple Terraform module that deploys resources in GCP to demonstrate how you can use Terratest to write automated tests for your GCP Terraform code. This module deploys a Compute Instance and gives that Instance a Name with the value specified in the instance_name variable. It also creates a Cloud Storage Bucket using the bucket_name and bucket_location variables. Note that the Compute Instance in this module doesn't actually do anything; it just runs a Vanilla Ubuntu 16.04 Image for demonstration purposes.
