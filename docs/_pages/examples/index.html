Check out below the list of examples with corresponding tests. To see some real-world examples of Terratest in action, check out some of our open source infrastructure modules.


Example Corresponding tests Description
Terraform Basic Example terraform_basic_example_test.go A simple "Hello, World" Terraform configuration.
Terraform HTTP Example terraform_http_example_test.go A more complicated Terraform configuration that deploys a simple web server that responds to HTTP requests in AWS.
Basic Packer Example packer_basic_example_test.go A simple Packer template for building an Amazon Machine Image (AMI) or Google Cloud Platform Compute Image.
Terraform Packer Example terraform_packer_example_test.go A more complicated example that shows how to use Packer to build an AMI with a web server installed and deploy that AMI in AWS using Terraform.
Terraform GCP Example terraform_gcp_example_test.go A simple Terraform configuration that creates a GCP Compute Instance and Storage Bucket.
Terraform remote-exec example terraform_remote_exec_example_test.go A terraform configuration that creates and AWS instance and then uses remote-exec to provision it.
Kubernetes Basic example kubernetes_basic_example_test.go kubernetes_basic_example_service_check_test.go A minimal Kubernetes resource that deploys an addressable nginx instance.
Kubernetes RBAC example kubernetes_rbac_example_test.go A Kubernetes resource config that creates a Namespace with a ServiceAccount that has admin permissions within the Namespace, but not outside.
Basic Helm Chart Example helm_basic_example_template_test.go helm_basic_example_integration_test.go A minimal helm chart that deploys a Deployment resource for the provided container image.
Packer Docker Example packer_docker_example_test.go A Packer template that creates an Ubuntu AMI with a simple web app. This template also creates a Docker image with the same web app installed.
Terraform ASG SCP Example terraform_scp_example_test.go A simple Terraform module that deploys resources in AWS to demonstrate how you can use Terratest to write automated tests for your AWS Terraform code.
Terraform AWS DynamoDB Example terraform_aws_dynamodb_example_test.go A simple Terraform module that deploys a DynamoDB table with server-side encryption, point in time recovery and a TTL (time to live) attribute.
Terraform AWS ECS Example terraform_aws_ecs_example_test.go A simple Terraform module that deploys a simple ECS service in AWS.
Terraform AWS Example terraform_aws_example_test.go A simple Terraform module that deploys resources in AWS. This module deploys an EC2 Instance and gives that Instance a Name tag with the value specified in the instance_name variable.
Terraform AWS Network Example terraform_aws_network_example_test.go A Terraform module that deploys a simple network setup to demonstrate how you can use Terratest to write automated tests for your AWS Terraform code. This module deploys two subnets (public and private) within one availability zone.
Terraform AWS RDS Example terraform_aws_rds_example_test.go A simple Terraform module that deploys a database instance (MySQL by default) in AWS.
Terraform AWS S3 Example terraform_aws_s3_example_test.go This module deploys an S3 Bucket and gives that Bucket a Name & Environment tag with the value specified in the tag_bucket_name and tag_bucket_environment variables, respectively.
Terraform Azure Example terraform_azure_example_test.go A simple Terraform module that deploys a Virtual Machine in Azure and gives that VM a Name tag with the value specified in the vm_name variable.
Terraform Backend Example terraform_backend_example_test.go A simple Terraform module that demonstrates how you can use Terratest to configure a Terraform Backend at test time.
Terraform GCP Managed Instance Group Example terraform_gcp_ig_example_test.go A simple Terraform configuration that deploys an Instance Group in GCP.
Terraform Redeploy Example terraform_redeploy_example_test.go This module deploys an Auto Scaling Group (ASG) in the AWS region specified in the aws_region variable and a Load Balancer to route traffic across the ASG.
Terraform SSH Example terraform_ssh_example_test.go This module deploys two EC2 Instances, one with a public IP, one with a private IP, in the AWS region specified in the aws_region variable. The EC2 Instances allow SSH requests on the port specified by the ssh_port variable.
Terraform SSH Password Example terraform_ssh_password_test.go The EC2 Instance allows SSH requests on the port specified by the ssh_port variable, and is configured with a user data script so that it will accept passwords for authentication.

Real-world examples


Example Description
Consul The repo contains a set of modules for deploying a Consul cluster on AWS using Terraform. Consul is a distributed, highly-available tool that you can use for service discovery and key/value storage.
Check test folder for Terratest tests.
Vault The repo contains a set of modules for deploying a Vault cluster on AWS using Terraform. Vault is an open source tool for managing secrets.
Check test folder for Terratest tests.
Nomad The repo contains a set of modules for deploying a Nomad cluster on AWS using Terraform. Nomad is a distributed, highly-available data-center aware scheduler.
Check test folder for Terratest tests.
Couchbase The repo contains a set of modules for deploying a Couchbase cluster on AWS using Terraform and Packer. Couchbase is a distributed NoSQL document database.
Check test folder for Terratest tests.
